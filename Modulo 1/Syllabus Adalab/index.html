¿Qué se aprende con Adalab?

Temario

Temario técnico

Durante este curso vamos a aprender a ser buenas programadoras de Front end.
Gracias a los profesores aprenderemos diferentes tipos de lenguajes que nos ayuden con la Maquetación de páginas web (HTML Y CSS).
Para crear páginas entretenidas y con las que se pueda interactuar, otro de los temas que aprenderás será el uso de Java Script

    Tipos de Lenguajes
  • "HTML" Lenguaje que sirve para estructurar el contenido de nuestras páginas web
  • "CSS" El cual, nos ayuda a definir los estilos de la web... qué tipo de fuente vamos a usar, cómo vana a estar separados los párrafos, los encabezados...
  • "Java Script" Se utiliza principalmente para crear páginas web dinámicas, con efectos de texto que aparece y desaparece, animaciones, botones...

    • Soft Skils

    Como muchas de las estudiantes de Adalab nunca han tenido contacto con el mundo laboral de la informática, se le enseñarán difertentes formas de afrontar las entrevistas de trabajo y, posteriormente, varias maneras de trabajar en equipo.
    Para que las Adalabers se acostumbren al ritmo de trabajo de las empresas, desde el primer día van a trabajar siguiendo la filosofía de Agile y el marco de trabajo Scrum, ambos muy usados en sus futuros trabajos.
